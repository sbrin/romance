# Searching Module (P0-01..P0-03)

## Назначение

- Сохраняет выбранную роль пользователя.
- Управляет очередью и матчингом по противоположному полу.
- Создает сессию в состоянии `PARTNER_FOUND`.
- Обрабатывает отмену поиска и уведомляет партнера.

## Контракты

- `POST /role`
- `POST /queue/join`
- `POST /queue/cancel`
- WS события: `partner_found`, `partner_cancelled`

## Обоснование

Изолирует core flow до старта сессии, чтобы UI и последующие модули
работали только с готовым результатом матчинга.
