# Partner Found (P0-03)

## Задача

Показать пользователю, что партнер найден, и подготовить к следующему шагу.

## Требования

- Отображает сообщение «Партнер найден!».
- Имеет кнопку «Отмена», возвращающую пользователя к выбору роли.
- Если партнер отменил поиск до старта сессии, показывает сообщение
  «Партнер отменил чат» и кнопку «Начать заново».
- Не содержит логики старта сессии (это P0-04).
- Реагирует только на внешнее состояние (события/ответы сервера).

## Обоснование

Разделяем «мэтч найден» и «старт сессии», чтобы не смешивать разные стадии
протокола и упростить дальнейшее развитие UX.
