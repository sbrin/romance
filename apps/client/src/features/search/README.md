# Search Screens (P0-03)

## Задача

Показать пользователю дефолтное состояние «начать поиск» и статусы поиска
(партнер найден / партнер отменил чат).

## Требования

- Отображает дефолтный экран с кнопкой «Начать поиск».
- Для найденного партнера показывает сообщение «Партнер найден!».
- Имеет кнопку «Отмена», возвращающую пользователя к экрану «начать поиск».
- Если партнер отменил поиск до старта сессии, показывает сообщение
  «Партнер отменил чат» и кнопку «Начать поиск».
- Не содержит логики старта сессии (это P0-04).
- Реагирует только на внешнее состояние (события/ответы сервера).

## Обоснование

Разделяем «старт поиска», «мэтч найден» и «старт сессии», чтобы не смешивать
разные стадии протокола и упростить дальнейшее развитие UX.
