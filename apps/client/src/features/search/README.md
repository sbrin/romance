# Search Screens (P0-03/P0-04)

## Задача

Показать пользователю дефолтное состояние «начать поиск», статусы поиска
(партнер найден / партнер отменил чат) и подтверждение старта сессии.

## Требования

- Отображает дефолтный экран с кнопкой «Начать поиск».
- Для найденного партнера показывает сообщение «Партнер найден!» и кнопку
  «Начать».
- После нажатия «Начать» отправляет `POST /session/start` и показывает статус
  «Жду партнера».
- После `session_started` переключается в состояние «Сессия начинается».
- Имеет кнопку «Отмена», возвращающую пользователя к экрану «начать поиск» и
  отменяющую сессию у обоих участников (через `partner_cancelled`).
- Если партнер отменил поиск до старта сессии, показывает сообщение
  «Партнер отменил чат» и кнопку «Начать поиск», сразу запускающую поиск.
- Реагирует только на внешнее состояние (события/ответы сервера).

## Обоснование

Разделяем «старт поиска», «мэтч найден» и «старт сессии», чтобы не смешивать
разные стадии протокола и упростить дальнейшее развитие UX.
