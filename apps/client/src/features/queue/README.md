# Queue Status (P0-02)

## Задача

Показать состояние ожидания партнера после успешного выбора роли и постановки
в очередь.

## Требования

- Отображает статус «Ищем партнера…».
- Визуально дает понять, что идет ожидание (пульс/спиннер).
- Имеет кнопку «Отмена», возвращающую пользователя к выбору роли.
- Не инициирует переходы самостоятельно, только отображает состояние.

## Обоснование

Очередь — отдельный UX-статус, поэтому держим отображение в собственной фиче
для последующих расширений (таймер, подсказки, ошибки).
