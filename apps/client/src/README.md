# App Shell

## Назначение

- Управляет глобальными UI-элементами, которые не принадлежат отдельному экрану.

## Требования

- Показывает глобальную кнопку «Выйти» в правом нижнем углу.
- Кнопка видима только в состояниях активного потока: поиск, мэтч, ожидание старта,
  активный шаг диалога.
- По нажатию отправляет `POST /queue/cancel` и возвращает в дефолтное состояние.
- При закрытии вкладки в состояниях поиска/подтверждения отправляет
  `POST /queue/cancel` через `sendBeacon` и сбрасывает локальный resume-state.
- При активной сессии сохраняет последний шаг диалога в localStorage и
  восстанавливает его при следующем открытии.
- При загрузке приложения проверяет сессию через `POST /session/resume` и
  восстанавливает состояние: `ACTIVE`, `FOUND` или `WAITING`.
- Видео шага отображается только при наличии текущего шага; при выходе видео пропадает.

## Обоснование

Глобальные элементы живут на уровне приложения, чтобы не дублировать логику
в отдельных экранах и сохранять единое поведение.
