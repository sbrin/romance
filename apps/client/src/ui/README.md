# UI Components

## Назначение

Набор небольших переиспользуемых UI-компонентов, соответствующих дизайн-плану
(крупные кнопки, стек статуса, safe-area рамка).

## Требования

- Компоненты не содержат сетевой логики.
- Все стили держатся в `App.css`/`index.css`.
- Компоненты пригодны для мобильного portrait UX.

## Обоснование

Выносим примитивы в `ui/`, чтобы фичи оставались компактными и легко
комбинировались между состояниями.
